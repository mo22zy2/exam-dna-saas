# Data Model: File Upload

**Date**: 2026-06-15

## Entity: File

**Table**: `files`

Extends the existing stub with user association and classification.

| Field | Type | Constraints | Description |
|-------|------|-------------|-------------|
| `id` | UUID | PK, default uuid4 | Unique file identifier |
| `user_id` | UUID (FK ‚Üí users.id) | NOT NULL, indexed | **NEW** ‚Äî owning user |
| `upload_session_id` | UUID (FK ‚Üí upload_sessions.id) | NULLABLE | Session this file was uploaded in |
| `filename` | VARCHAR(255) | NOT NULL | Original filename (displayed in UI) |
| `storage_key` | VARCHAR(512) | NOT NULL, unique | Internal storage path |
| `file_size` | INTEGER | NOT NULL | Size in bytes |
| `classification` | VARCHAR(20) | NOT NULL | **NEW** ‚Äî "exam" or "lecture" |
| `mime_type` | VARCHAR(50) | NOT NULL, default "application/pdf" | **NEW** ‚Äî validated MIME type |
| `status` | VARCHAR(20) | NOT NULL, default "uploaded" | **NEW** ‚Äî uploaded/processing/ready/failed |
| `created_at` | TIMESTAMPTZ | NOT NULL | Upload timestamp |
| `updated_at` | TIMESTAMPTZ | NOT NULL | Last update timestamp |

**Relationships**:
- `user` ‚Üí `User` (M:1) ‚Äî each file belongs to one user
- `upload_session` ‚Üí `UploadSession` (M:1) ‚Äî optional session link

**Indexes**:
- `ix_files_user_id` on `user_id`
- `ix_files_user_id_status` composite on `(user_id, status)`
- Unique on `storage_key`

## Entity: UploadSession

**Table**: `upload_sessions`

Extends the existing stub.

| Field | Type | Constraints | Description |
|-------|------|-------------|-------------|
| `id` | UUID | PK, default uuid4 | Unique session identifier |
| `user_id` | UUID (FK ‚Üí users.id) | NOT NULL, indexed | Owning user |
| `status` | VARCHAR(20) | NOT NULL, default "pending" | pending/uploading/completed/failed |
| `file_count` | INTEGER | NOT NULL, default 0 | **NEW** ‚Äî total files in this session |
| `created_at` | TIMESTAMPTZ | NOT NULL | Session creation time |
| `updated_at` | TIMESTAMPTZ | NOT NULL | Last update time |

**Relationships**:
- `user` ‚Üí `User` (M:1)
- `files` ‚Üí `File` (1:M)

**Status Transitions**:
```
pending ‚Üí uploading ‚Üí completed
                   ‚Üò failed
```

## Validation Rules

- `file_size`: 1 ‚â§ value ‚â§ 52,428,800 (50 MB in bytes)
- `classification`: one of `"exam"`, `"lecture"` (lowercase, enum)
